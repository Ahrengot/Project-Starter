Project starter

You now have a base template including Twitter Bootstrap & Underscore.js

For your convinience you also have Sass, Gulp and CoffeeScript available. Livereloading has been set up and Bower will let you pull in all of the libraries and frameworks you need. Have fun!

— To get started run these commands:

  1. $ npm install
  2. $ bower install
  3. $ gulp
