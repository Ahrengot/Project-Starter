## Project starter with the following

* Webpack
* React.js
* Redux + Dev tools
* React Router
* PostCSS loader for JS components
* Hot module replacement
* SVG Sprites
* Browserify (easy debugging on multiple devices)
* Bootstrap 4 alpha (Sass version)
